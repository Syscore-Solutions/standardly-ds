Directions

Sprint S1 creative core · three directions o1 / o2 / o3 for the Tue 2026-07-14 team call · definitions pending S1-T03

Each direction is a semantic remap + component styling treatment over the same fixed primitives — proving the token architecture can carry three distinct visual personalities without touching a single raw value. The switcher in the rail drives the preview surfaces below; the mechanism is wired, but all three map identically until Naman's direction research (S1-T03) lands.

07.1

The rules of the game

Locked — every direction obeys

  • Purple stays the brand primary (confirmed 2026-07-10)
  • Primitives are fixed — no new ramps, fonts or scales
  • Single light theme; no dark mode
  • WCAG 2.1 AA contrast; visible focus rings
  • Status never by color alone — icon or shape always pairs

Free — where directions differ

  • Semantic mapping — which primitive each role points at
  • Radius & density — sharp-tight through soft-airy
  • Shadow / elevation treatment
  • Motion character (restrained per brief §10)
  • Composition — how the same components assemble a page
