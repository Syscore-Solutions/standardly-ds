The Core Challenge

We need to deliver a premium, accessible UI for complex certification workflows — but we don't have a frontend engineering team.


If design and code are disconnected, everything we design will break in implementation, or take months to build.

The Token Pipeline

1. tokens.json

The single source of truth. W3C DTCG format. Synced automatically to Figma via Tokens Studio.

2. Automated Build

A 112-line script generates CSS custom properties instantly. No manual handoff.

3. Semantic Mapping

Primitives are remapped to semantic roles (e.g. `color.bg.brand` = `purple.50`).

4. Components

React components consume variables. Proven today in Storybook.

Locked Decisions

  • Brand Primary: Purple (locked across all apps)
  • Tech Stack: React + TypeScript + Radix UI (Phase 2)
  • Styling: CSS Variables + Vanilla CSS (No Tailwind)
  • Accessibility: WCAG 2.1 AA minimum built into tokens
